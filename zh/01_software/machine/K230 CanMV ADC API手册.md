## 1. 概述

ADC提供了channel配置、使能、关闭和读取数据功能

## 2. API描述

ADC提供了一个类ADC，实现了三个函数，init(),deinit(),value()

### 2.1 类 machine.ADC

【描述】

通过指定的参数新建一个 ADC 对象。

【语法】

```python
from machine import ADC
ADC = ADC(channel, start=True)
```

【参数】

| 参数名称        | 描述                          | 输入/输出 |
|-----------------|-------------------------------|-----------|
| channel  |  通道号（0~5）。            | 输入      |
| start  |  是否使能adc,默认为True            | 输入      |

【返回值】

| 返回值  | 描述                            |
|---------|---------------------------------|
| 0       | 成功                          |
| 非 0    | 失败 |

【注意】

无

【举例】

无

【相关主题】

无
